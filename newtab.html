Add Quick Access

How to use

  • Add a task — click + New Task in the sidebar.
  • Edit a task — double-click its title.
  • Complete a task — click the round checkbox. Done tasks move to Completed.
  • Favorite — click the star on a task row.
  • Delete a task — hover the row, click the trash icon.
  • Add a quick link — click the + Add tile, set name, URL and logo.
  • Edit / delete a link — right-click the tile.
  • Reorder links — drag a tile onto another.
  • Backup your data — open Settings and use Export. Import restores it on any device so you never lose tasks.

Settings

Export your tasks and quick links to a file as a backup, then import it anytime to restore — so you never lose a task.
